비디오 태그를 활용한 영상 가져오기

웹브라우저에서 인식 가능한 영상 파일 타입 : mp4, obv, webm

비디오 태그를 구성하는 주요 속성들

1. source 태그를 활용하여 여러가지 형식의 영상을 작성한다.

2. 브라우저가 열리면서 자동 재생이 되기 위해서는 video 태그 내부에 autoplay 속성을 작성한다. (단, iOS에서는 자동 재생을 위해서 playsinline 이라는 속성도 함께 작성되어야 함)

3. loop : 반복재생 / muted : 음소거 활성화 / controls : 재생과 일시정지 기능을 활성화 시킴
